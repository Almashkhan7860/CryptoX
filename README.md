⚡ Overview

CryptoX is an advanced Python-based cybersecurity toolkit designed for learning cryptography, password hashing, encryption techniques, and CTF challenges.

It combines multiple security utilities into a single interactive terminal application, allowing users to:

✅ Identify Hash Types
✅ Generate Secure Hashes
✅ Test Password Strength Against Hashes
✅ Encrypt & Decrypt Files
✅ Encrypt & Decrypt Text
✅ Learn Practical Cryptography Concepts

✨ Features

🔍 Hash Identifier

Detect common hash formats instantly:

- MD5
- SHA1
- SHA256
- SHA512
- BCRYPT

---

🔨 Hash Generator

Generate hashes from any text input:

- MD5
- SHA1
- SHA256
- SHA512
- BCRYPT

Perfect for:

- Security Labs
- Password Auditing Practice
- CTF Challenges
- Learning Hash Algorithms

---

💥 Hash Verification Tool

Test hashes against custom or built-in wordlists.

Includes support for:

- MD5
- SHA1
- SHA256
- SHA512
- BCRYPT

Useful for:

- Security Education
- Password Security Demonstrations
- Capture The Flag Exercises
